title: File
---

**FileTools** enable an Assistant to read and write files on the local file system.

## Example

The following assistant will generate an answer and save it in a file.

```python cookbook/tools/file_tools.py
from phi.assistant import Assistant
from phi.tools.file import FileTools

assistant = Assistant(tools=[FileTools()], show_tool_calls=True)
assistant.print_response("What is the most advanced LLM currently? Save the answer to a file.", markdown=True)
```

## Toolkit Params

<ResponseField name="base_dir" type="Path">
  Specifies the base directory path for file operations.
</ResponseField>
<ResponseField name="save_files" type="bool" default="True">
  Determines whether files should be saved during the operation.
</ResponseField>
<ResponseField name="read_files" type="bool" default="True">
  Allows reading from files during the operation.
</ResponseField>
<ResponseField name="list_files" type="bool" default="True">
  Enables listing of files in the specified directory.
</ResponseField>

## Toolkit Functions

<ResponseField name="save_file">
  Saves the contents to a file called `file_name` and returns the file name if
  successful.
</ResponseField>
<ResponseField name="read_file">
  Reads the contents of the file `file_name` and returns the contents if
  successful.
</ResponseField>
<ResponseField name="list_files">
  Returns a list of files in the base directory
</ResponseField>
